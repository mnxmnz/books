## Getting Started

#### Installation

1. Clone the Repo

```shell
git clone https://github.com/mnxmnz/Book-Report.git
```

```shell
cd Book-Report
```

2. Install Project Packages

```shell
yarn
```

3. Run the Project (Dev Mode)

```shell
yarn start

http://localhost:3000/
```
